Color palette: desert gold and ochre, Persian Gulf turquoise, lamp-warm amber, dusty rose sunset, deep shadow. Painterly historical style, split composition with unifying desert landscape, atmospheric lighting. No text on the image.

## Tips for Best Results

- **Aspect ratio**: Use 2:3 (portrait/book cover format). In Midjourney: `--ar 2:3`
- **Style modifiers to try**: Add `oil painting`, `concept art`, `illustration`, `photorealistic` depending on the look you want
- **Quality**: In Midjourney add `--q 2` or `--s 750` for higher stylization
- **No text**: All prompts specify no text — add title/author text in a design tool afterward (Canva, Photoshop, etc.)
- **Variations**: If a result is close but not perfect, use the AI tool's variation/remix features to refine
- **Consistency**: To keep a consistent series look, save a result you like and use it as a style reference for subsequent covers
- **For Midjourney specifically**: Append `--style raw` for less stylized, more photographic results, or `--niji` for more illustrated/anime-influenced results
